## For the end user

Run your estate or letting agency's finances from one HTML file. 15 tabs cover CMP, rent, deposits, contractors, landlord disbursements, fees, sales, properties, tenancies, clients, agents, firm P&L, expenses, and Q&A.

### Key features

| Tab | What it does |
|---|---|
| Dashboard | KPI overview + compliance checklist |
| CMP | Client Money Protection · daily reconciliation · scheme status |
| Rent | Rent ledger · arrears tracking · due/paid/overdue status |
| Deposits | Housing Act 2004 · 30-day protection tracking · DPS/MyDeposits/TDS |
| Contractors | Maintenance payments · VAT · client account vs landlord direct |
| Landlord | Monthly statements · rent − fees − maintenance − VAT = net |
| Fees | Agency fees · setup/monthly/renewal/checkout · fixed or % |
| Sales | Sales commission · agreed % · pending/received |
| Firm P&L | Rolling 12-month · revenue − expenses − overheads · VAT threshold |

### Key rules

- **CMP**: mandatory for letting agents holding client money (SI 2019/386)
- **Deposit protection**: Housing Act 2004 — 30 days, 1-3x penalty
- **VAT threshold**: £90k (2025-26) — standard rate on agency fees
- **Redress**: PRS or TPO membership required (ERRA 2013)
- **AML**: HMRC supervision under MLR 2017
